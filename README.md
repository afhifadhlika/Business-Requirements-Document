**Summary** :

Satudata is a digital-based data collection project referred to as Big Data, where the data collection process through to data analysis is carried out digitally and eventually can be utilized. This implementation can be done in a shorter time and offers higher efficiency compared to conventional data collection, with the expectation that the data obtained will be more accurate and credible.

**Objective** :

1. Improving data management with Satudata Big Data
2. Reducing data collection time with Satudata Big Data
3. Enhancing data processing with Satudata Big Data
4. Improving data analysis with Satudata Big Data
5. Maintaining data quality with Satudata Big Data
6. Securing data in Satudata Big Data
7. Implementing data with Satudata Big Data
